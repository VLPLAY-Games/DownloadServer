🎯 Key Features

File Upload - Paste a link into the input field and click "Add Task"

Download Management - Use the Pause/Resume, Restart, Delete buttons

Progress Tracking - Monitor the progress bar and download speed

File Download - After completion, click the "Download" button

🎮 Task Management

⏸️ Pause - Temporarily suspend the download

▶️ Resume - Continue a paused download

🔄 Restart - Start the download over

🗑️ Delete - Remove the task and the downloaded file

⚙️ Settings

🌐 Language - Switch between Russian and English

🔐 Account - Log in to save download history

💾 Autosave - Tasks are saved even after page reload

❓ Frequently Asked Questions

Q: What file formats are supported?
A: All formats - video, music, documents, archives

Q: Maximum file size?
A: No limits, depends on available server space

Q: How long are files stored?
A: Files are stored until you delete them
